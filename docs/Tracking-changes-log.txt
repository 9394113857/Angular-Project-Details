1. Using --legacy-peer-deps
One quick way to bypass the issue is to install the dependencies while ignoring peer dependency conflicts using the --legacy-peer-deps flag.

Run this command:
bash:-
npm install bootstrap --legacy-peer-deps

What this does: This command installs Bootstrap 
but ignores the peer dependency conflicts that npm is reporting. 
It can often solve issues caused by mismatched peer dependencies 
without requiring you to fix the underlying version mismatch.
